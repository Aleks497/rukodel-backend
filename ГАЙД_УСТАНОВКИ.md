# 🚀 ПОЛНЫЙ ГАЙД: Установка проекта «РукоДел» с нуля

---

## 📋 ЧТО НУЖНО УСТАНОВИТЬ (один раз)

1. Python 3.10+  →  https://www.python.org/downloads/
2. MySQL 8.0+    →  https://dev.mysql.com/downloads/installer/
3. PyCharm CE    →  https://www.jetbrains.com/pycharm/download/

---

## ШАГ 1 — Установка Python

1. Скачайте Python с https://www.python.org/downloads/
2. Запустите установщик
3. ✅ ОБЯЗАТЕЛЬНО отметьте «Add Python to PATH»
4. Нажмите Install Now
5. Проверка в PowerShell:
   python --version
   # Должно вывести: Python 3.10.x или выше

---

## ШАГ 2 — Установка MySQL

1. Скачайте MySQL Installer: https://dev.mysql.com/downloads/installer/
2. Выберите "MySQL Server Only" или "Developer Default"
3. При установке задайте пароль для root (ЗАПОМНИТЕ его!)
4. После установки запустите MySQL Workbench или командную строку MySQL

---

## ШАГ 3 — Создание папки и скачивание файлов проекта

Создайте папку для проекта:

    mkdir D:\Projects\rukodel
    cd D:\Projects\rukodel

Скачайте все файлы из чата (кнопка Download в предпросмотре)
и распакуйте в папку D:\Projects\rukodel

Структура должна быть такой:

    rukodel/                ← папка проекта
    ├── manage.py
    ├── requirements.txt
    ├── .env
    ├── rukodel/
    │   ├── __init__.py
    │   ├── settings.py
    │   ├── urls.py
    │   └── wsgi.py
    ├── shop/
    │   ├── __init__.py
    │   ├── apps.py
    │   ├── models.py
    │   ├── views.py
    │   ├── admin.py
    │   └── migrations/
    │       └── __init__.py
    └── templates/
        └── shop/
            ├── base.html
            ├── index.html
            ├── login.html
            ├── register.html
            ├── cart.html
            ├── checkout.html
            ├── orders.html
            └── product_detail.html

---

## ШАГ 4 — Открыть проект в PyCharm

1. Запустите PyCharm
2. File → Open → выберите папку D:\Projects\rukodel → OK
3. Нажмите "Trust Project"

---

## ШАГ 5 — Создать виртуальное окружение в PyCharm

1. File → Settings (Ctrl+Alt+S)
2. Project: rukodel → Python Interpreter
3. Нажмите шестерёнку ⚙ → Add Interpreter → Add Local Interpreter
4. Выберите Virtualenv Environment → New
5. Location: D:\Projects\rukodel\venv
6. OK

---

## ШАГ 6 — Установить зависимости

Откройте Terminal в PyCharm (Alt+F12) и выполните:

    pip install -r requirements.txt

Подождите окончания установки (1-2 минуты).

---

## ШАГ 7 — Создать базу данных MySQL

В Terminal PyCharm выполните:

    mysql -u root -p

Введите пароль root (тот, что задавали при установке MySQL).
Затем выполните эти команды по одной:

    CREATE DATABASE rukodel CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
    CREATE USER 'rukodel_user'@'localhost' IDENTIFIED BY 'rukodel_pass';
    GRANT ALL PRIVILEGES ON rukodel.* TO 'rukodel_user'@'localhost';
    FLUSH PRIVILEGES;
    EXIT;

---

## ШАГ 8 — Настроить файл .env

Откройте файл .env в PyCharm и убедитесь, что данные совпадают с тем, что вы указали в MySQL:

    DB_NAME=rukodel
    DB_USER=rukodel_user
    DB_PASSWORD=rukodel_pass
    DB_HOST=localhost
    DB_PORT=3306

Если вы задали другой пароль — измените DB_PASSWORD.

---

## ШАГ 9 — Создать папку migrations

В Terminal:

    mkdir shop\migrations
    type nul > shop\migrations\__init__.py

---

## ШАГ 10 — Применить миграции (создать таблицы в БД)

    python manage.py makemigrations
    python manage.py migrate

Должно появиться много строк "OK" — это таблицы создаются в MySQL.

---

## ШАГ 11 — Создать администратора

    python manage.py createsuperuser

Введите данные (пароль не отображается — вводите вслепую):

    Username: admin
    Email: admin@rukodel.ru
    Password: admin123
    Password (again): admin123

Если Django скажет "This password is too common" — нажмите y и Enter.

---

## ШАГ 12 — Запустить сервер

    python manage.py runserver

Должно появиться:

    Django version 4.2.8
    Starting development server at http://127.0.0.1:8000/
    Quit the server with CTRL-BREAK.

## 🔁 КАК ЗАПУСКАТЬ КАЖДЫЙ РАЗ

Каждый раз когда хотите запустить проект:

1. Откройте PyCharm → откройте проект
2. Откройте Terminal (Alt+F12)
3. Активируйте venv (PyCharm делает это автоматически)
4. Запустите сервер:

       python manage.py runserver

5. Откройте браузер: http://127.0.0.1:8000/

## ❌ ЧАСТЫЕ ОШИБКИ

ОШИБКА: "No module named 'rukodel.wsgi'"
РЕШЕНИЕ: Убедитесь, что файл rukodel/wsgi.py существует

ОШИБКА: "Access denied for user"
РЕШЕНИЕ: Проверьте данные в .env файле

ОШИБКА: Порт 8000 занят
РЕШЕНИЕ: python manage.py runserver 8080

ОШИБКА: "Table doesn't exist"
РЕШЕНИЕ: python manage.py migrate

## ✅ ИТОГОВАЯ СТРУКТУРА URL

    /orders/             — Мои заказы
    /product/<id>/       — Страница товара с отзывами
    /admin/              — Административная панель
